  • CentoOS7 minimal 出现ping: unknown host www.baidu.com解决办法：

    • 修改 （vi /etc/sysconfig/network-scripts/ifcfg-eth0） 

    • 重启网络 （systemctl restart network.service（或 service network restart））

    • 查看 （cat /etc/resolv.conf ） 

      注意：不要在 /etc/resolv.conf 直接添加图片中的内容，CentOS7 重启网络会把 /etc/sysconfig/network-scripts/ifcfg-eth0 的DNS内容重新赋于到 /etc/resolv.conf 中

      此时可以尝试 ping www.baidu.com（若仍然ping不通执行继续执行）

    • 差看路由表（netstat -rn） 

    • 增加网关（route add default gw 192.168.175.1）
