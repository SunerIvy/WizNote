1.寻找入口点
    web程序都是从web.xml开始的，然后寻找到入口程序，由“spring启动过程”可得知，首先会初始化ContextLoaderListener
<listener-class>com.baosight.iplat4j.core.web.listener.PlatformContextLoaderListener</listener-class>
但是从该文件中未找到，所以推断是自己做了自定义封装，故去找类似的PlatformContextLoaderListener，进入该类会发现确实如此
public void contextInitialized(ServletContextEvent event) {
    PlatApplicationContext.initProjectConfig();
    super.contextInitialized(event);
}
自定义了一个PlatApplicationContext.initProjectConfig(),由这个入口就能找到所有的初始化内容
